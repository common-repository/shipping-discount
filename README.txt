== Description ==

Want to make a strikeout price for shipping? It's easy to use the shipping discount plugin, all you have to do is set the shipping discount you want.

== Features ==

* Easy without the need for a lot of configuration, just set it in the default WooCommerce coupon menu.
* There is an additional discount type "Shipping Discount" in the coupon settings.
* There are 3 types of discounts: Free Shipping, Fixed Amount, and Percentage.
* There is detailed discount information in the WP Admin.
* There is an additional meta in Shipping.

== Installation ==

Installation and configuration is very easy!

1. Download and activate the Shipping Discount plugin.
2. Activate coupons in the WooCommerce menu > Settings > General > Enable coupons.
3. Create a Coupon in Marketing > Coupons > Add Coupon.
4. Name the coupon.
5. Select Discount Type "Shipping Discount".
6. Select Shipping Discount Type: "Free", "Fixed" or "Percentage".
7. Click "Publish"
